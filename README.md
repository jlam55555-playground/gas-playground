# gas-playground
x86-64 GNU (AT&T-style) Assembly playground

### ViM setup
To get some nicer alignment items in ViM, you can run the following when ViM is open:
```vim
:so PATH/TO/THIS/DIR/asm.vim
```

### Compilation steps
```bash
gcc [filename] -no-pie
```
